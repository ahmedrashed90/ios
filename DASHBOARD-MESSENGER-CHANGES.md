# تعديلات الداش بورد

- كروت مبيعات الكاش ومبيعات التمويل وخدمة العملاء تظل بجانب بعضها.
- الضغط على كارت القسم يفتح الحالات، والضغط عليه مرة ثانية يغلقها.
- كل حالة تفتح وتقفل بشكل مستقل.
- إظهار جميع عملاء الحالة بدون حد 60 عميل.
- عرض العملاء كقائمة قريبة من تطبيقات المحادثة.
- داخل المحادثة يظهر اسم العميل في رأس الشات.
- الضغط على اسم العميل يفتح بياناته لتعديلها أو تغيير الحالة وحفظها.
- عند إغلاق بيانات العميل تعود مساحة الشات كاملة.

لم يتم تغيير مسارات Firebase أو منطق التوزيع أو التقارير.
